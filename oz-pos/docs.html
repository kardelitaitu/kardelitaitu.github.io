Introduction

Welcome to the oz-pos enterprise documentation. oz-pos is a fast, modular, offline-first point-of-sale platform developed to run reliably on modern register systems while maintaining cloud synchronization.

Installation

The compiled oz-pos binaries are provided as a single daemon executable. Download the system binary corresponding to your server distribution and place it in your service execution bin path.

To initialize default configurations, run the launch command:
./oz-pos --init

Lua Configuration

Configurations are loaded via the config.lua file. This white-label script lets you specify shop configurations, custom colors, metadata headers, and active system modules dynamically without having to recompile the binaries.

Writing Discounts

Custom discounts are configured dynamically using standard Lua script syntax. Write your customized discount functions inside the calculate_discount trigger, which takes a cart parameters table and computes the markdown balance.
