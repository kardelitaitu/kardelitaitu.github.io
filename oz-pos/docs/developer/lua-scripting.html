Lua Configuration & White-Label Customization

The embedded Lua engine executes configurations at launch and dynamic checkout calculations during execution.

White-Label Schema

Define branding assets and colors directly in the global config table inside config.lua. 

config.white_label = {
    brand_color = "#ea580c",
    receipt_header = "Welcome to Our Premium Store!",
    show_logo = true
}

Dynamic Price Rules

You can write standard Lua functions to control shopping cart promotions. The core invokes calculate_discount(cart) on transaction change. 

function calculate_discount(cart)
    -- Apply 10% discount on transaction value if over $100
    if cart.total > 100.00 then
        return cart.total * 0.10
    end
    return 0.0
end
