Rust Core API Reference

The back-end server runs on a modular Rust daemon structured around key architectural traits.

StoreManager Implementation

The entry point is the StoreManager struct. It controls checkout threads, registers background listeners, and locks database transactions. 

pub struct StoreManager {
    db: SqlitePool,
    cashier_sessions: Arc<Mutex<HashMap<Uuid, CashierSession>>>,
    active_terminal_id: String,
}

Build Requirements

To compile the daemon project locally, ensure you have the Rust stable toolchain installed:
cargo build --release -p oz-pos-daemon
