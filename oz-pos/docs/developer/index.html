Developer Overview

Welcome to the oz-pos Developer Portal. Use the navigation sidebar to explore technical topics, compile settings, and platform customization APIs.

Our application consists of a high-performance Rust core integrated with a Tauri front-end wrapper to control terminal registers, printers, and barcode readers. Custom white-label rules are scripted dynamically via Lua.
