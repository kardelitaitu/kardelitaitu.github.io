Tauri Desktop UI Wrapper

oz-pos uses Tauri to compile native desktop applications for Windows, macOS, and Linux registers. Tauri runs a lightweight webview referencing our HTML/JS frontend and bridges system calls directly to our local Rust daemon.

Hardware Listeners Bridge

We bridge physical devices (receipt printers, customer pole displays, barcode scanners) using Tauri's IPC commands. 

// Tauri command for raw receipt printer execution
#[tauri::command]
fn print_receipt(raw_escpos: Vec<u8>) -> Result<(), String> {
    let mut printer = SerialConnection::open("/dev/ttyUSB0")?;
    printer.write(&raw_escpos)?;
    Ok(())
}

Running Locally

To run the Tauri development environment locally, invoke:
npm run tauri dev
